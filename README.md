# CS2 Steam Profile Stats

[![Support me on Boosty](https://img.shields.io/badge/boosty-50%E2%82%BD/month-green.svg?logo=boosty)](https://boosty.to/mah_boi)

A small static HTML/CSS/JS page for quickly opening CS2-related Steam profile stats services from one place.

## Features

- Steam profile link input.
- Local search history with saved Steam profile avatars.
- Embedded tabs for supported services.
- Middle mouse button redirects directly to the selected service.
- Static GitHub Pages deployment via GitHub Actions.

## Supported Services

- [leetify.com](https://leetify.com/)
- [csst.at](https://csst.at/)
- [csrep.gg](https://csrep.gg/)
- [fastgg.pro](https://fastgg.pro/trust-factor-checker/)
